# Tecnologias Utilizadas 
* **Lenguajes de Programacion:** PHP, Python, CSS, HTML
* **Desarrollo Web (Frontend):** HTML5
* **Entorno de Servidor Local:** XAMPP (Apache)
* **Entorno de Desarrollo:** Visual Studio Code

# Competencias Desarrolladas 
* **Desarrollo de Aplicaciones Web:** Creación de interfaces funcionales y dinámicas mediante la integración de HTML, CSS y PHP, conectando la lógica de servidor con la experiencia del usuario.
* **Pensamiento Algorítmico y Automatización:** Resolución de problemas complejos utilizando estructuras de control, funciones y programación orientada a objetos en Python y PHP.
* **Gestión de Servidores Locales y Bases de Datos:**Manejo de bases de datos locales, lectura y escritura en archivos de texto, así como organización de datos para su uso eficiente en aplicaciones.
**Fundamentos de Ingeniería Informática:**Estructuración de proyectos, modularización de código, documentación y mantenimiento de soluciones de software escalables.
